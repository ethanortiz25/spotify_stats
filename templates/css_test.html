Margins vs Padding Demonstration

The boxes below show the difference between margins and padding:

MARGIN Example

This red box has 40px margin (outside space) and no padding. Notice the space OUTSIDE the black border.

PADDING Example

This cyan box has 40px padding (inside space) and no margin. Notice the space INSIDE the black border.

BOTH Margin + Padding

This green box has BOTH 30px margin (outside) and 30px padding (inside). The margin creates space between this and other elements, while padding creates space between the border and this text.
