Wordle Solver

I created this website using Azure Cloud Services, and a Python Flask server. On this website, the user is told what word to guess on Wordle. After inputting the word into whatever website they are playing in, the user reports which colors were given back onto my website. After sending a request to the server, a python script is run on the server which tells the user which word they should guess next. This process is repeated until the word is guessed correctly. In my testing the algorithm that I came up with solves the Wordle on average in 3.5 guesses.

Point of Sale System

I created this website using Azure Cloud Services, and a Python Django server. I worked as a full stack developer in a team of 5 people using the Agile method to develop a mock POS system for Piada. In this I was responsible for creating CRUD operations for our database. This included menu items, ingredients, users, and orders. Through this I learned not only how to use Django and PostgreSQL but also how to work in a team using the Agile and Scrum methods. This also included the ability for users to place mobile orders with the menu populated from our database. It allows for control of the menu with creation, deletion, and editing of menu items for managers. The websites also allows for the creation of admin users which can create, view, elevate, and delete user accounts. This allows for users to be created the same for staff and customers with the admin being able to elevate a user to staff, manager, or admin. We also impletemented several API's including google oauth and google translate.
